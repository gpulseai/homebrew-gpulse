# Homebrew Tap for gpulse

Official Homebrew tap for [gpulse](https://gpulse.ai) — real-time GPU monitoring with leak detection and OOM prediction.

## Install

```bash
brew tap gpulseai/gpulse
brew install gpulse
```

## Update

```bash
brew update
brew upgrade gpulse
```

## Uninstall

```bash
brew uninstall gpulse
brew untap gpulseai/gpulse
```
